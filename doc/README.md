**MyClashShell**为Ubuntu设计的clash命令行软件，本质上就是围绕clash写的一些脚本，能够帮你自动配置命令行版本的clash并提供更简单的使用方法，无需图形化界面支持，方便一些没有图形化界面的嵌入式设备使用。

支持架构 

- AMD64
- ARMv8
- ARMv7a
